Enceintes
Vos Sonos / Symfonisk, jointes par IP. L'état est rafraîchi en continu par le réconciliateur.
Webradios
Flux HTTP continus à diffuser. Privilégier des URL http (le https natif Sonos est capricieux).
Programmation
Tranches horaires : quelle webradio sur quelle enceinte, quels jours. Si une enceinte était éteinte au démarrage, elle est rattrapée automatiquement.
